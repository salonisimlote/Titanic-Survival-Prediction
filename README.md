# Titanic Survival Prediction using Machine Learning

## Overview

This project implements an end-to-end data science pipeline for predicting passenger survival on the Titanic dataset using machine learning.

The project includes:

- ETL Pipeline
- Data Cleaning
- Exploratory Data Analysis
- Database Schema Design
- Machine Learning
- Feature Importance Analysis
- Hyperparameter Tuning
- Survival Prediction

---

## Dataset

Titanic Dataset

Source:
https://www.kaggle.com/competitions/titanic
